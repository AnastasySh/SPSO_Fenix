
# Проект: «Fenix»

Данная ИС, а именно web-приложение предназначено для членов отряда "СПСО ФЕНИКС" с входом строго по логину и паролю. 
Целевым назначением web-приложения является обучение членов отряда навыкам и компетенциям спасателя разного уровня, а так же тестирование и отчетность руководству отряда и вышестоящим отрядам.
